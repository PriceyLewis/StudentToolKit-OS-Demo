# Store Listing + Compliance Checklist

## Required contact and links
- Support email address is active.
- Privacy Policy URL is publicly accessible.
- Terms of Use URL is publicly accessible.

## Safe store description language
Use wording like:
- planning
- tracking
- goal setting
- habit reminders

Avoid risky claims like:
- guaranteed results
- makes money
- improves health

## Data safety declarations
For the current local-only model, declare accurately:
- No data collected
- No data shared

Re-check these if you add:
- analytics
- crash reporting
- cloud backup
- account sync
- third-party SDKs

## Final verification
- Policy URLs open correctly from the app.
- Onboarding and core flows work with no account.
- Dashboard links and privacy flows match the current app behavior.
