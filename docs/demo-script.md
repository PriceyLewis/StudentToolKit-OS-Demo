# Demo Script

## Positioning

Student Toolkit OS is a local-first mobile product mock built with Expo Router and TypeScript. It is designed to show product thinking, multi-screen UX, persistence, and planning workflows rather than backend scale.

## 60-Second Walkthrough

1. Open onboarding and explain that the app frames a student's week across academics, training, habits, income goals, and career growth.
2. Enter the dashboard and point out the aggregate score, target tracking, coaching prompts, and local-first data model.
3. Open one planner screen such as Revision or CV to show structured inputs, instant scoring, and export flows.
4. Show habits or weekly review to demonstrate repeat-use behavior and reflection loops.
5. Close on analytics to show that the app connects multiple workflows into a single progress narrative.

## Talking Points

- Built as a cohesive product concept, not just isolated screens
- Uses React Context plus AsyncStorage for persistent local state
- Includes export, notifications, backup, and review flows to make the mock feel like a real product
- Structured so authentication, sync, and cloud analytics could be added later without rewriting the entire app

## What To Say If Asked "What Would You Build Next?"

- Authentication and cross-device sync
- Real calendar integration and smarter reminders
- More robust analytics and goal forecasting
- A cleaner production design system and component library
- Automated testing around key planner and persistence flows
